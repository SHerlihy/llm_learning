# LLM Learing

## Description

LLM Learning is a website providing a curated repository of learning resources all related to LLMs.

## Motivation

After provisioning books so my colleagues are able to learn foundational LLM concepts, my colleagues expressed they wanted to know more about LLMs.
I asked them what facets of LLMs they were interested in and recieved a wide variety of answers from "prompt engineering" to "ethics and governance".
Recognising the subject area as rapidly evolving and multi faceted I decided I needed a platform that would meet these demands.
I have created LLM learning to use a tag based discovery feature so my colleagues can tailor the resources they discover to their own interest and for ease of addition for new facets as they emerge in this evolving subject.

## Usage

1. Visit [LLM Learning](https://d2gru0zoilocfa.cloudfront.net)
2. Click the tags that interest you
3. Enjoy learning about LLMs
